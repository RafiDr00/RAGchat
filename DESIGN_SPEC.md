# RAGchat - Academic Professional UI Spec

## Design System - Grayscale Only

### Colors
- Background: #0b0b0b
- Surface: rgba(255,255,255,0.04)
- Border: rgba(255,255,255,0.08)
- Text Primary: #ffffff
- Text Secondary: rgba(255,255,255,0.6)
- Caret: rgba(255,255,255,0.6)

### Typography
- Headline: font-semibold tracking-tight
- Body: font-regular
- UI: font-mono tracking-wider opacity-40

### Effects (Minimal)
- Shadow: 0 1px 2px rgba(0,0,0,0.4), 0 8px 24px rgba(0,0,0,0.6)
- Text Shadow: 0 2px 12px rgba(0,0,0,0.6)

NO:
- Green/emerald colors
- Glow effects
- Gradient text
- Text stroke
- Backdrop blur (excessive)
